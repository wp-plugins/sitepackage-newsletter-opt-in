This plugin provides a newsletter opt-in form that is connected to sitepackage://

== Description ==

To use this plugin you need to register for the sitepackage:// newsletter system on http://www.sitepackage.de. There is a free version available.

sitepackage:// is a web based newsletter system to create, send and analyse personalized e-mailings and their spread across social media.

This plugin provides a fast and easy way to integrate a newsletter opt-in form into your website. The form consists of a title, an introductory text, an e-mail input field, a button and a privacy notice text. All texts and descriptions are editable. When setting up the form you also have to set your sitepackage:// project domain and form id.
On submit the value of the e-mail input field, the project domain and the form id will be sent to an external sitepacke:// server to process the opt-in.

== Installation ==

Follow these steps to install the plugin:

1. To install from the plugin repository click the install button otherwise copy the file "sitepackage-newsletter-opt-in.php" into the folder "/wp-content/plugins/".
1. Activate the plugin in the (Plugins) "Installed plugins" panel.
1. Open the (Design) "Widgets" panel.
1. Drag and drop the "sitepackage:// Newsletter Opt-In" widget in one of the widget areas.
