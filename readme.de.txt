Dieses Plugin stellt ein Widget für ein Newsletter-Anmeldeformular zur Verfügung, das an sitepackage:// angebunden wird.

== Description ==

Zur Nutzung dieses Plugins ist eine Registrierung für das sitepackage:// Newsletter-System auf http://www.sitepackage.de nötig. Die Registrierung für die Freeware-Version ist kostenlos.

sitepackage:// ist ein webbasiertes Newsletter-System für Erstellung, Versand und Auswertung personalisierter E-Mailings und deren Verbreitung über soziale Medien.

Dieses Plugin bietet eine schnelle und einfache Möglichkeit ein Newsletter-Anmeldeformular in Ihre Webseite zu integrieren. Das Formular besteht aus Titel, Einleitungstext, einem Eingabefeld für die E-Mail-Adresse, einer Schaltfläche und einem Privatsphärehinweis. Alle Texte und Beschriftungen können Sie frei definieren.

== Installation ==

Zum Installieren des Plugins führen Sie folgende Schritte durch:

1. Laden Sie die Datei "sitepackage-newsletter-opt-in.php" in den Ordner "/wp-content/plugins/".
1. Aktivieren Sie das Plugin im Administrationsbereich unter dem Menüpunkt "Plugins".
1. Wählen Sie den Menüpunkt "Design" und dann "Widgets".
1. Ziehen Sie das Widget "sitepackage:// Newsletter Opt-In" in den gewünschten Widget-Bereich.
